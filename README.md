# Event Management App

A cross-platform mobile application built with Flutter and Firebase 
for managing departmental events in educational institutions.

## Features
- Role-based access control (4 user roles — Admin, Faculty, Student, Guest)
- QR code-based attendance tracking
- Real-time event dashboard powered by Firestore
- Push notifications via Firebase Cloud Messaging (FCM)
- Automated PDF and Excel report generation
- Supports Android, iOS, Web, Windows, macOS, Linux from one codebase

## Tech Stack
Flutter · Dart · Firebase Firestore · Firebase Auth · 
Firebase Cloud Messaging · Supabase · Provider · QR Flutter

## Built by
4-member team — IV Semester B.Tech CSE project at SVPCET, Nagpur

## Setup
1. Clone the repo
2. Run `flutter pub get`
3. Connect your Firebase project via `firebase.json`
4. Run `flutter run`
